# 元信号参考

读取市场价格做预测时，以下 18 种信号模式信息量最高。按问题需要选用，不必全部覆盖。

## 信号目录

### 1. 单一行业商品
只服务一个行业的原材料 = 该行业活动的纯信号。
- 螺纹钢 → 中国建筑 | 钯金 → 汽车产量 | 锂 → 电动车 | GPU租赁价 → AI算力需求

### 2. 困境债回收定价
龙头债券在面值一折以下 = 市场认为资产减值远未结束。25-30分以上 = 有底部买家进场。
- 债券比股票诚实，参与者几乎全是机构。

### 3. 上游设备商订单
交付周期 2-3 年、取消成本极高的订单 = 最硬的前瞻信号。
- 前瞻性排序：设备商订单(3年) > 晶圆厂产能(2年) > 终端营收(1年) > 应用层融资(6-12月)

### 4. "超预期但下跌"
业绩超预期但股价跌 = 所有好消息已定价完毕，边际预期只能走差。增长见顶的领先信号。
- 反过来："低于预期但上涨" = 最坏时期可能已过。

### 5. CapEx vs 股价分裂
宣布大幅增加投资但股价跌 = 市场认为投入产出比在恶化。投资周期从"扩张期"进入"质疑期"。

### 6. 两种商品价差分化
两种相关但服务不同需求的商品在分化 = 结构性转型信号。
- 铜涨+铁矿跌 = 新经济扩张、旧经济收缩
- 金银比极端 = 贵金属内部投机狂热
- 近月vs远月(backwardation/contango) = 供需紧张的时间预期

### 7. 银行 P/B
P/B 远低于 1.0x = 市场预期的隐性坏账。P/B 回升 = 坏账出清接近尾声。

### 8. 超长周期基础设施投资
市场为 5-10 年建设周期的基础设施疯狂投资 = 需求趋势是结构性的、十年级别的。
- 核电、海底光缆、电网扩容、军工产能

### 9. 风投集中度
融资总额涨但轮次降、大额轮占比升 = 行业"晚期繁荣"，从广撒网转向赌赢家。

### 10. 龙头估值折价
基本面无可挑剔但估值持续低于同行 = 非经营性风险被定价。折价幅度 = 风险大小。
- 台积电 PE 18x vs 英伟达 39x → 差额 ≈ 台海风险定价

### 11. 不可逆资本配置
企业/政府做出大额不可逆投资来对冲风险 = 对风险的真实判断。
- 看三个属性：金额大不大？能不能退？目标日期是什么？目标日期 ≈ 风险窗口
- 台积电搬 30% 而非 100% = 定价"中断"而非"毁灭"

### 12. 保险精算定价
精算师用统计模型定价，不受情绪干扰。保费暴涨 10x+ = 模型输入发生实质变化。
- 航运战争险、台海保费、网络安全险

### 13. 单一用途商品价格崩塌
信号 #1 的反面。价格暴跌 = 供需关系已逆转，不需要分析报告。
- GPU租赁价跌 75-85% → AI算力供给过剩

### 14. 高管减持节奏
减持本身不一定看空，但节奏变化是信号。多家公司高管同时加速减持 = 行业性信号。

### 15. 股权转债务融资
行业投资规模超过现金流、大规模举债 = 在杠杆化信念。投资周期晚期特征。
- 发债规模从年均到当年的倍数 = 杠杆化速度

### 16. 市场宽度 / 主题轮动
龙头集体跑输等权指数 = 资金从主题撤退但不离开市场。泡沫"慢性失血"模式。

### 17. 杠杆资金集中度
杠杆资金过度集中在单一主题 = 保证金追缴风险。集中度本身是系统性脆弱性指标。

### 18. 官方统计 vs 市场价格背离
有行政管控的官方数据和纯市场定价出现大幅背离 = 永远信市场。背离幅度 = 多少真相被压住。

## 时间维度排列

按每种信号定价的未来窗口排列，判断信号之间的矛盾时首先看它们在哪个时间位置。

```
短期 (3-12月)   ← 预测市场合约、VIX/MOVE、价格反应模式(#4)、高管减持(#14)
中期 (1-3年)    ← 龙头营收共识、CapEx计划(#5)、风投集中度(#9)、杠杆集中度(#17)
长期 (3-10年)   ← 设备商订单(#3)、不可逆资本配置(#11)、超长基建投资(#8)
超长期 (10年+)  ← 核电站建设、军费结构性扩张、人口结构
```

## 信号分裂时的处理

信号分裂时，不投票取多数，而是：

1. **看时间维度** — 短期悲观+长期乐观 ≠ 矛盾，= S曲线正在弯折
2. **看"两件事同时发生"** — 旧经济日本化+新经济活跃可以共存于同一经济体
3. **"方向对但时机不对"** — 长期信号看多但短期过热 → 结论不是"买/不买"而是"等回调"
